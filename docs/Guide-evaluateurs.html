Ce que contient la clé USB

  • index.html — l'application d'évaluation (à ouvrir d'un double-clic).
  • Guide-evaluateurs.html — ce document.
  • Les fichiers d'évaluation enregistrés au fur et à mesure (un par étudiant).

Réglage à faire UNE seule fois sur l'ordinateur

Pour que chaque évaluation s'enregistre toute seule sur la clé, on règle le dossier de téléchargement du navigateur sur la clé USB.

Safari (Mac)
  1. Menu Safari → Réglages…
  2. Onglet Général
  3. « Emplacement de téléchargement » → Autre… → choisir la clé USB
Chrome / Edge
  1. Menu ⋮ → Paramètres → Téléchargements
  2. « Emplacement » → choisir la clé USB
  3. Désactiver « Toujours demander où enregistrer »

Déroulé d'une évaluation

  1. Brancher la clé USB, puis double-cliquer sur index.html.
  2. Sélectionner l'étudiant, ajuster les notes sur le radar, compléter les sous-items.
  3. Rédiger les commentaires (points positifs + axes d'amélioration).
  4. Signer dans le cadre, puis cliquer sur « Oui, valider ».
  5. ➡️ Le fichier de l'évaluation s'enregistre automatiquement sur la clé. Aucune autre action.

Ouvrir un fichier d'évaluation enregistré

Les fichiers sont protégés par mot de passe. En double-cliquant sur un fichier, saisir le code :

0405

⚠️ À respecter

  • La clé doit rester branchée pendant toute l'évaluation et au moment de valider.
  • Ne pas vider ni renommer les fichiers de la clé : ce sont les sauvegardes.
  • Travailler dans un seul onglet à la fois. Éviter la navigation privée.
  • En fin de session, vérifier que les fichiers des étudiants sont bien présents sur la clé.
Application autonome — aucune donnée n'est envoyée sur Internet. Les évaluations restent sur la clé USB.
